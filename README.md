
Taller integrador de arboles binarios escrito en C#

## Requisitos

- [.NET 10.0 SDK](https://dotnet.microsoft.com/download)

## Ejecución

Estando en el root del proyecto
```bash
dotnet run
```

## Funcionalidades

- **Insertar** documentos y carpetas (con conteo de comparaciones)
- **Buscar** documentos (con conteo de comparaciones)
- **Actualizar** nombre de un documento (Eliminación + Inserción, preserva tipo carpeta/archivo)
- **Eliminar** documentos (informa si era hoja, nodo con un hijo, o nodo con dos hijos)
- **Mostrar árbol** en formato ASCII con conectores (├──, └──, │)
- **Recorridos**: Preorden, Inorden, Postorden, BFS
- **Altura** del árbol
- **Datos de prueba**: opción para cargar 14 nodos predefinidos
